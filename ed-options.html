ed: standard editor

Some of ed's commands:

Command Description
e [file] edit [file].
w write changes.
q quit without saving.
(X,Y)d delete lines in range (X,Y).
(1,$)g/RE/p get lines matching Regular Expression
in the whole file (1,$) and print them.
(.,$)s/RE/re/ substitute RE with replacement, starting
from the current line (.) up to the EOF ($).
