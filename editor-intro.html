What's a text editor?
My definition:
Software that allows the user to make complete
and thoughtful text changes.
Examples:
  • WYSIWYG Rich Text document editor (e.g. LibreOffice Writer)
  • Simple text file editor (e.g. Notepad++)
  • Feature-rich scriptable text transformator tool
What doesn't count:
  • Text formatter (only applies predefined transformations)
  • Format converter (-||-)
